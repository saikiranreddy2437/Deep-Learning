# Data Folder

Place your dataset files here.

Required files:
- `train.tsv`
- `val.tsv`
- `test.tsv`

These files are NOT included in this repository because of copyright restrictions from SemEval-2019 Task 3.

Format:
Each TSV file must contain the following columns:

- turn1
- turn2
- turn3
- label

Example row:
Hi   How are you?   I'm fine   happy
